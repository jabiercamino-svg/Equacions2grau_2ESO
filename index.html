T1

Equacions incompletes — Tipus 1

Forma \(ax^2 + c = 0\) · Aïlla \(x^2\), divideix per \(a\) i aplica l'arrel quadrada

Exemple resolt Equació incompleta Tipus 1: \(ax^2 + c = 0\)

Resol: \(3x^2 - 48 = 0\)

Aïlla el terme amb \(x^2\)
\(3x^2 = 48\)
Divideix pel coeficient \(a\)
\(x^2 = \dfrac{48}{3} = 16\)
Arrel quadrada als dos membres
Si \(x^2 = k\) amb \(k > 0\) → \(x = \pm\sqrt{k}\) \(x = \pm\sqrt{16} = \pm4\)
Solucions: \(x_1 = 4\)  i  \(x_2 = -4\)

⚠️ Si \(k < 0\) no hi ha solució real. Si \(k = 0\) la única solució és \(x = 0\).

T2

Equacions incompletes — Tipus 2

Forma \(ax^2 + bx = 0\) · Treu \(x\) factor comú i aplica la regla del producte zero

Exemple resolt Equació incompleta Tipus 2: \(ax^2 + bx = 0\)

Resol: \(2x^2 - 10x = 0\)

Treu \(x\) com a factor comú
\(x(2x - 10) = 0\)
Regla del producte zero
\(x = 0 \quad\text{o}\quad 2x - 10 = 0\)
Resol cada cas
\(x_1 = 0 \qquad 2x = 10 \;\Rightarrow\; x_2 = 5\)
Solucions: \(x_1 = 0\)  i  \(x_2 = 5\)

⚠️ Les equacions T2 sempre tenen dues solucions: \(x=0\) i una altra.

Equacions completes — Fórmula general

Primer amb \(a=1\), després amb \(a \neq 1\). Sempre la mateixa fórmula.

Fórmula general $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
Exemple resolt Fórmula general — de \(a=1\) a \(a \neq 1\)

APARTAT A — \(a = 1\): \(\;x^2 + 3x - 10 = 0\)

Identifica els coeficients i calcula \(\Delta\)
\(a=1,\; b=3,\; c=-10 \;\Rightarrow\; \Delta = 9+40 = 49\)
Aplica la fórmula
\(x = \dfrac{-3 \pm \sqrt{49}}{2} = \dfrac{-3 \pm 7}{2}\)
Calcula les dues solucions
\(x_1 = \dfrac{-3+7}{2} = 2 \qquad x_2 = \dfrac{-3-7}{2} = -5\)
\(x_1 = 2\)  i  \(x_2 = -5\)

APARTAT B — \(a \neq 1\): \(\;2x^2 - 7x + 3 = 0\)

Identifica els coeficients i calcula \(\Delta\)
\(a=2,\; b=-7,\; c=3 \;\Rightarrow\; \Delta = 49-24 = 25\)
Aplica la fórmula (atenció: \(2a = 4\))
\(x = \dfrac{7 \pm \sqrt{25}}{4} = \dfrac{7 \pm 5}{4}\)
Calcula les dues solucions
\(x_1 = \dfrac{7+5}{4} = 3 \qquad x_2 = \dfrac{7-5}{4} = \dfrac{1}{2}\)
\(x_1 = 3\)  i  \(x_2 = \dfrac{1}{2}\)

Δ

El discriminant

Determina quantes solucions té l'equació sense resoldre-la

Fórmula \(\Delta = b^2 - 4ac\)
  • ▶\(\Delta > 0\): l'equació té 2 solucions reals diferents
  • ▶\(\Delta = 0\): l'equació té 1 solució doble
  • ▶\(\Delta < 0\): l'equació no té solució real
Exemple resolt Càlcul del discriminant

Calcula \(\Delta\) de \(2x^2 - 4x + 2 = 0\) i digues quantes solucions té.

Identifica els coeficients
\(a=2, \quad b=-4, \quad c=2\)
Calcula \(\Delta\)
\(\Delta = (-4)^2 - 4 \cdot 2 \cdot 2 = 16 - 16 = 0\)
Interpreta
\(\Delta = 0\) → l'equació té una solució doble.
\(\Delta = 0\) → 1 solució doble

Resol: \((x + 3)(x - 1) = 0\)

Identifica el tipus

\(x_1 = -3 \qquad x_2 = 1\)
\(x_1 = -3\)  i  \(x_2 = 1\)

Planteja l'equació, resol-la i interpreta la solució en el context del problema

Exemple resolt Com plantejar i resoldre un problema

La base d'un rectangle és 5 m més llarga que l'alçada. L'àrea és 36 m². Quines són les dimensions?

Defineix la incògnita
Sigui \(x\) l'alçada (m). La base és \(x+5\).
Planteja l'equació
\(x(x+5) = 36 \;\Rightarrow\; x^2 + 5x - 36 = 0\)
Resol
\(\Delta = 25+144 = 169 \;\Rightarrow\; x = \dfrac{-5 \pm 13}{2}\) \(x_1 = 4 \qquad x_2 = -9\)
Interpreta i descarta valors no vàlids
\(x_2 = -9\) no té sentit (mesura negativa). Alçada \(= 4\) m, base \(= 9\) m.
Les dimensions del rectangle són 4 m × 9 m.

\(n(n+1)=110 \;\Rightarrow\; n^2+n-110=0\)
